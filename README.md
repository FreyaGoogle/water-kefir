# 💧 Water Kefir Assistent

Een Nederlandse web-app om je te helpen bij het brouwen van water kefir.

## Functies

- **F1** – Bereken ingrediënten voor de eerste fermentatie op basis van je kefirkorrels
- **F2** – Kies smaakstof en fermentatietijd, met drukwaarschuwingen
- **Planner** – Plan je kefir-cyclus op ochtend- of avondritme
- **Check** – Gezondheidscheck voor je batch
- **FAQ** – Veelgestelde vragen over water kefir

## Starten

```bash
npm install
npm run dev       # → http://localhost:5173
npm test          # → alle tests uitvoeren
npm run build     # → productie build in dist/
```

## Technologie

- React 18 + TypeScript
- Vite
- Vitest
